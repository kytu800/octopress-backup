Web Design Concepts Index

接觸 Web Design 常會不知道世界上有什麼好玩的新東西。
於是這裡羅列了一些網站，只有簡單名詞跟介紹，目的是提供幾個關鍵字讓各位更深入學習！

Editor

Sublime Text 2
http://www.sublimetext.com/
務必安裝 package install ， 相當方便的套件管理!
http://wbond.net/sublime_packages/package_control/installation

Basic

HTML 教學

http://www.w3schools.com/html/default.asp

CSS 教學

http://css.1keydata.com/tw/

Javascript

直接從 jQuery 入門會比較輕鬆 XD (個人意見
http://jquery.com/

想寫得更快嗎？

ZenCoding

跟編輯器配合下能夠快速產生 CSS 跟 HTML
http://code.google.com/p/zen-coding/

SCSS/SASS

提供一種編譯成 CSS 的語法，具有變數、函數、巢狀的概念
http://sass-lang.com/

Compass

建立於 SCSS/SASS 的 Library
http://compass-style.org/

LESS

類似 SCSS ，提供 Client 端編譯
http://lesscss.org/

CoffeeScript (Actually I don’t use it.)

提供一種編譯成 Javascript 的語法
http://coffeescript.org/

CSS layout Framework

960 grid

http://960.gs/

blueprint

http://www.blueprintcss.org/

Responsive CSS Framework

Bootstrap

個人偏愛，原始碼為 LESS
http://twitter.github.com/bootstrap/

Foundation

類似 Bootstrap，原始碼為 SCSS
http://foundation.zurb.com/

http://jquerymobile.com/

JQuery UI

http://jqueryui.com/

MVC

各種MVC比較
http://addyosmani.github.com/todomvc/

Backbone.js

目前最火紅
http://backbonejs.org/

Underscore.js

不是 MVC ，作者跟 Backbone.js 、 Coffee Script 為同一位

並有常用的 template 功能建議大家可以深入學習
http://underscorejs.org/
